## 1. Executive Summary

This document is the official completion report and invoice for the custom software system built for **Jay Dee Sales Corporation (JDSC)**.

The main goal of this project was to replace slow, manual paperwork with a smooth digital system. It helps the office staff and field workers stay connected, tracks orders in real-time, and automates daily tasks. The complete system includes a central admin control panel, an easy-to-use Android app for employees, a secure online portal for your customers, and a handy desktop tool to quickly split combined invoice PDFs by client.

## 2. Core System Architecture

The platform is built using a modern, multi-tier stack designed to be secure, fast, and resilient:

1. **Centralized Backend Engine (API):** Built on the enterprise-grade **Laravel** framework, serving as the central brain. It handles the database, business logic, calculations, notifications, and accounting ledgers.
2. **Administrator & Employee Android App:** Developed with **Flutter** for cross-platform speed and responsive design. This is the daily app used by company admins and employees.
3. **Client Interface Web Portal:** A lightweight, secure web application allowing your customers to view their order history and financial statements instantly without needing to remember usernames and passwords.
4. **Web Admin Dashboard:** A central panel where administrators can upload spreadsheets (such as starting balances or invoices) and manage system-wide settings.
5. **PDF-Pal (Invoice Splitting Desktop Utility):** A standalone desktop application built with Python to automate the splitting of consolidated, multi-customer invoices into individual sanitized PDFs.

## 3. System Specifications & Accomplished Features

### A. Customer Relationship & Smart Sales Management (CRM)

- **Custom Customer Pricing:** Standard products can be assigned unique prices on a _per-customer_ basis. When placing an order, the system automatically retrieves that customer's specific rate.
- **Tally Spreadsheet Integration:** You can upload Excel files directly into the system to import new customers or update legacy outstanding balances from tools like Tally.
- **Automatic Call Log CRM:** The Android app monitors calls. When an admin hangs up after calling a customer, the app opens a prompt allowing you to save notes from the conversation and schedule a follow-up alarm.
- **Exact CRM Reminders:** Set alarms inside the app for follow-up dates. The app schedules native Android exact alarms to alert you precisely when it is time to call a client back.
- **Easy Statement Sharing:** Admins can generate and share public statements or invoices with clients via native sharing tools on their phones.

### B. Intelligent Billing & FIFO Ledger Engine

- **First-In, First-Out (FIFO) Payment Matching:** When a customer pays you, the system automatically matches the payment against their oldest unpaid bills. It marks older bills as "Paid" first and rolls any leftover money down to newer bills.
- **Advance Payments Ledger:** If a customer pays more than their total outstanding balance, the system automatically logs the surplus as an "Advance Payment" (credit balance) to offset future orders.
- **Cached Balance Performance:** Customer balances are recalculated daily and cached. This means looking up a customer's statement is lightning-fast and doesn't slow down the database.
- **Archived Accounts Filter:** You can mark older, inactive, or archived accounts (by prefixing their name with `"Z "`) to automatically hide them from outstanding balance sheets.

### C. Workforce Operations & Piece-Rate Payroll

- **Two Employment Profiles:** The system supports standard **Salary-based** employees (paid by hours worked) and **Contract-based** workers (paid by items/pieces produced).
- **Digital Timecard Punching:** Employees can punch in and out directly inside the Android app, tracking their worked hours down to the minute.
- **Pieces Staging & Equal Splitting:** Contract teams can log completed piece-rate work (e.g. products made) and select the teammates involved. The admin reviews this staging queue, and upon approval, the system splits the earnings **equally** among the listed team members.
- **Leave Management System:** Employees can request leaves through the app. The backend automatically prevents overlapping leave requests.
- **Automated Monthly Salary Sheets:** The system compiles opening balances, timecard earnings, contract payouts, salary advances (debits), and payments (credits) for any month into a single, easy-to-read salary statement.
- **Sunday Autoclock Scheduler:** The system automatically logs a standard 8-hour shift every Sunday for salary employees (unless they are registered on approved leave) to save manual admin work.

### D. Order Pipelines & Automated Communication

- **Order Status Pipeline:** Track orders through four distinct stages: `Created` &rarr; `Processing` &rarr; `Shipped` &rarr; `Completed`.
- **Zero-Login Customer Web Portal:** Customers receive a secure web link (UUID link). By clicking this link, they can instantly view their order progress and financial ledger history. No email, password, or sign-up forms required.
- **Push Notifications for Admins:** Whenever a new order is placed by employees or logged into the system, all administrators instantly receive a Firebase Push Notification on their phones. Clicking the notification opens the order directly.

### E. PDF-Pal — Invoice Splitting Desktop Utility

- **Smart Invoice Splitting:** Automatically groups multi-page invoices by page numbers and dynamically extracts customer names using regex-based pattern matching (prefixed with `M/s` or `Ms`).
- **File Naming & Sanitization:** Converts customer names to a clean lowercase, underscore-delimited format, strips OS-forbidden characters, truncates names to 50 characters, and automatically resolves duplicate file conflicts.
- **Themed Desktop GUI:** Built with Python's standard `tkinter` framework, using a custom cream and dark-grey color scheme, dynamic custom fonts, real-time progress indicators, and native file explorer integration.
- **Tech Stack & Architecture:** Built with Python 3.11+, using `PyPDF2` for PDF text extraction/generation, and packaged using `PyInstaller` into standalone executables.
- **Desktop Distribution:** Bundled and configured a standalone desktop installer for Windows (`PDF-Pal.exe`).
