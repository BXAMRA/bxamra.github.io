# Punjab Educare

### Empowering Teachers & Students with Technology

## Features

- **Comprehensive Curriculum** — Access study materials from Nursery to 10+2 classes aligned with Punjab School Education Board syllabus.
- **Quiz & Assessment** — Interactive quizzes with randomized questions, progress tracking, and detailed performance analytics.
- **Multilingual Certificates** — Generate certificates in English, Punjabi, and Hindi for course completion and achievements.
- **Teacher Portal** — Dedicated portal for teachers to submit questions, manage content, and track student progress.
- **Secure & Private** — Role-based authentication with secure data handling compliant with India's DPDP Act 2023.
- **Cross-Platform** — Available on Android with iOS support, featuring responsive design and offline capabilities.

## Tech Stack

**Mobile Apps** — Flutter & Dart with Material Design for cross-platform Android and iOS _coming soon_.\
`Flutter` `Dart` `Material Design`

**Desktop Apps** — Tauri with Vue.js for lightweight, native-performance desktop applications.\
`Tauri` `Vue.js` `Tailwind` `Shadcn-Vue`

**API Backend** — Laravel optimised with Laravel Octane for high-performance REST API with role-based authentication.\
`Laravel` `PHP` `Laravel Octane`

**Database & Cache** — MySQL as the primary database with Redis for caching and performance optimisation.\
`MariaDB` `Redis`
