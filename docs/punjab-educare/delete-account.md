# Account Deletion Request

::: warning
- Only **teacher accounts** can be deleted (students don’t have accounts)
- Deletion is **irreversible**
- Processing time: up to **4 weeks** after email verification
- Approved content you submitted (e.g. questions) may remain in our database
:::

## Request by email

Email: **accounts@punjabeducare.co.in**  
Send from your **registered teacher email** and include:

- Registered email
- Full name
- Reason (optional)

## Email template

Subject:

```
Account Deletion Request
```

Body:

```
Please delete my Punjab Educare teacher account.

Registered Email: [your email]
Full Name: [your name]
Reason (optional): [reason]
```
